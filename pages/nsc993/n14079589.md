  外出闯荡多挣钱，

  可是二老却一夜之间撒手而去，
  <br/>
  儿子再也没有了这个机会。
  <br/>
  那村东头张家的姑娘，
  <br/>
  说好了我多挣钱回来娶你，
  <br/>
  你怎么就独自离我而去？
  <br/>
  只能在梦中看到你的倩影。
  <br/>
  生死两茫茫，
  <br/>
  比刀郎的花妖更令人心疼。

  人民生命财产怎么这么不值钱？
